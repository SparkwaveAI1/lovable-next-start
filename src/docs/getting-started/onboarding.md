# Onboarding Flow

This guide provides a detailed walkthrough of the Sparkwave onboarding process. If you're stuck at any step, this is your reference.

## Overview

Sparkwave's onboarding consists of 4 main phases:

1. **Account Creation** — Sign up and verify
2. **Business Setup** — Configure your business profile
3. **Integration Connection** — Link your existing tools
4. **Agent Configuration** — Set up your first AI agents

## Phase 1: Account Creation

### Sign Up

When you first visit Sparkwave, you'll see the sign-up form:

- Enter your **email address**
- Create a **password** (minimum 8 characters, must include a number)
- Accept the Terms of Service

### Email Verification

After signing up:

1. Check your inbox for a verification email
2. Click the verification link
3. You'll be automatically logged in and redirected

> **Tip:** Check your spam folder if you don't see the email within 2 minutes.

## Phase 2: Business Setup

### Basic Information

You'll be asked to provide:

| Field | Description | Example |
|-------|-------------|---------|
| Business Name | Your company name | "Peak Fitness Studio" |
| Industry | Your business category | Fitness & Wellness |
| Timezone | For scheduling | America/New_York |

### Business Details

Optional but recommended:

- **Website URL** — Helps Rico understand your business
- **Phone Number** — For SMS agent verification
- **Address** — For location-based features

## Phase 3: Integration Connection

### Wix Integration

If you use Wix:

1. Click **Connect Wix**
2. Log in to your Wix account
3. Grant Sparkwave permission to access:
   - Contacts
   - Bookings (if applicable)
   - Pricing Plans (if applicable)
4. Select which site to connect

### Payment Processing

Connect your payment processor for:

- Automated payment reminders
- Subscription management
- Revenue reporting

### Communication Channels

Set up outbound messaging:

- **Email** — Verify your sending domain
- **SMS** — Verify your phone number

## Phase 4: Agent Configuration

### Choosing Your First Agent

We recommend starting with **one agent** and adding more as you get comfortable. Best first choices:

| Agent | Best For | Why Start Here |
|-------|----------|----------------|
| Lead Follow-up | Service businesses | Highest ROI, immediate results |
| Booking Reminder | Appointment-based | Reduces no-shows quickly |
| Review Request | Any business | Easy to measure success |

### Agent Settings

For each agent, you'll configure:

- **Activation** — Turn the agent on/off
- **Timing** — When the agent should act
- **Messaging** — Customize the copy
- **Escalation** — When to involve a human

## Post-Onboarding Checklist

Before considering setup complete:

- [ ] All agents show "Healthy" status
- [ ] Test contact receives automated message
- [ ] Rico responds to basic questions
- [ ] Health dashboard shows green indicators
- [ ] Activity feed is logging events

## Common Onboarding Issues

### "Integration failed to connect"

- Ensure pop-ups aren't blocked
- Try a different browser
- Check that you have admin access to the tool

### "Agent won't activate"

- Verify the required integration is connected
- Check that all required fields are filled
- Ensure you have contacts matching the agent's criteria

### "No activity showing"

- Add a test contact to trigger automation
- Wait 2-3 minutes for processing
- Check Health Monitoring for any errors
