# Reports & Analytics

Sparkwave's reporting gives you clear visibility into your automation performance. Understand what's working, identify opportunities, and prove ROI.

## Dashboard Overview

The main dashboard shows key metrics at a glance:

- **Active Automations** — Number of agents currently running
- **Tasks Today** — Total automated actions taken
- **Success Rate** — Percentage of successful completions
- **Time Saved** — Estimated hours of manual work avoided

## Available Reports

### Activity Summary

**What it shows:** Overview of all automation activity

- Total tasks by agent
- Success/failure breakdown
- Peak activity times
- Trending vs. previous period

**Best for:** Daily/weekly check-ins

### Agent Performance

**What it shows:** Deep dive into each agent's metrics

- Tasks processed
- Success rate
- Average response time
- Conversion metrics (where applicable)

**Best for:** Optimizing agent configuration

### Conversion Funnel

**What it shows:** Lead progression through your funnel

```
New Leads → Contacted → Responded → Converted
   100         95          42          28
```

- Drop-off points identified
- Conversion rate at each stage
- Time-to-conversion metrics

**Best for:** Understanding lead quality and process

### Health Report

**What it shows:** System reliability metrics

- Uptime percentage
- Error rate trends
- Integration health history
- Performance over time

**Best for:** Technical oversight, SLA tracking

### ROI Calculator

**What it shows:** Value of your automation

- Tasks automated (vs. manual time estimate)
- Cost per task (vs. employee cost)
- Monthly time savings
- Annual ROI projection

**Best for:** Proving value to stakeholders

## Using Reports

### Date Range Selection

All reports support flexible date ranges:

- **Today** — Current day
- **This Week** — Monday to now
- **This Month** — 1st to now
- **Last 30 Days** — Rolling window
- **Custom** — Pick start and end dates

### Filtering

Narrow down reports by:

- **Agent** — Specific agent or all
- **Status** — Success, failed, all
- **Contact Segment** — Leads, customers, etc.

### Exporting

Export any report:

1. Run the report with desired filters
2. Click "Export" button
3. Choose format:
   - **PDF** — For sharing/printing
   - **CSV** — For spreadsheet analysis
   - **JSON** — For integrations

## Key Metrics Explained

### Success Rate

```
Success Rate = Completed Tasks / Total Tasks × 100
```

**Healthy range:** 95-99%

**If below 95%:**
- Check error logs for patterns
- Review data quality
- Verify integrations

### Response Rate

```
Response Rate = Contacts Who Replied / Contacts Reached × 100
```

**Industry benchmarks:**
- SMS: 15-25%
- Email: 5-15%

### Conversion Rate

```
Conversion Rate = Converted / Total Leads × 100
```

Measured at each funnel stage to identify drop-offs.

### Time Saved

Calculated based on:
- Number of tasks automated
- Estimated manual time per task (default: 5 min)
- Your configured hourly rate

## Scheduled Reports

### Setting Up Auto-Reports

Receive reports automatically:

1. Go to Reports → Scheduled
2. Click "New Schedule"
3. Configure:
   - Report type
   - Frequency (daily/weekly/monthly)
   - Recipients
   - Format

### Available Schedules

| Frequency | Delivered | Best For |
|-----------|-----------|----------|
| Daily | 8 AM | Operations monitoring |
| Weekly | Monday 9 AM | Management review |
| Monthly | 1st of month | Stakeholder updates |

## Troubleshooting

### Report Not Loading

1. Check date range (not too broad?)
2. Verify you have data in selected period
3. Try a different browser
4. Contact support if persists

### Data Looks Wrong

1. Check filter settings
2. Verify date range
3. Ensure timezone is correct
4. Compare with Activity feed

### Export Failed

1. Try a smaller date range
2. Check browser download settings
3. Try a different format
4. Clear browser cache
