# Common Issues & Solutions

Quick fixes for frequently encountered problems.

## Login & Authentication

### Can't Log In

**Symptoms:** Login button doesn't respond, or you see an error message.

**Solutions:**
1. Clear your browser cache and cookies
2. Try an incognito/private window
3. Check that your email is correct (it's case-sensitive)
4. Use the "Forgot Password" link to reset
5. Ensure you're using a supported browser (Chrome, Firefox, Safari, Edge)

### Session Keeps Expiring

**Symptoms:** You're logged out frequently.

**Solutions:**
1. Check your browser's cookie settings
2. Disable browser extensions that might block cookies
3. Ensure your system clock is accurate
4. Contact support if the issue persists

## Dashboard & Navigation

### Dashboard Won't Load

**Symptoms:** Blank page or endless loading spinner.

**Solutions:**
1. Refresh the page (Ctrl+R or Cmd+R)
2. Clear browser cache
3. Check your internet connection
4. Try a different browser
5. Check [status.sparkwaveai.app](https://status.sparkwaveai.app) for outages

### Missing Menu Items

**Symptoms:** Expected navigation options aren't visible.

**Solutions:**
1. Your account may have limited permissions
2. Some features require a higher plan tier
3. Contact your administrator to check access levels

---

## Automations & Tasks

### Automation Not Running

**Symptoms:** Scheduled tasks don't execute.

**Solutions:**
1. Verify the automation is set to **Active**
2. Check the trigger conditions are met
3. Review the activity log for error messages
4. Ensure connected integrations are working
5. Check that you haven't hit rate limits

### Tasks Stuck in "Pending"

**Symptoms:** Tasks show as pending indefinitely.

**Solutions:**
1. Check if the required integration is connected
2. Verify API credentials haven't expired
3. Look for errors in Mission Control
4. Try canceling and recreating the task

---

## Integrations

### Integration Won't Connect

**Symptoms:** OAuth fails or connection errors appear.

**Solutions:**
1. Ensure pop-ups aren't blocked
2. Check that you're using the correct account
3. Verify the third-party service is operational
4. Try disconnecting and reconnecting
5. Clear browser cache and try again

### Data Not Syncing

**Symptoms:** Changes in connected services don't appear in Sparkwave.

**Solutions:**
1. Check the last sync timestamp in Settings
2. Manually trigger a sync if available
3. Verify permissions on the integration
4. Review for conflicts or duplicate records
5. Contact support for sync issues lasting over 24 hours

---

## AI Agents & Rico

### Rico Not Responding

**Symptoms:** Chat messages get no response.

**Solutions:**
1. Check your internet connection
2. Refresh the page
3. Try a shorter, simpler message
4. Check if you've hit usage limits
5. Wait a moment and try again (service may be busy)

### Agent Giving Wrong Answers

**Symptoms:** AI responses are inaccurate or unhelpful.

**Solutions:**
1. Provide more context in your question
2. Review and update the agent's knowledge base
3. Check agent configuration settings
4. Report persistent issues to help improve the AI

## Performance

### Slow Page Loads

**Symptoms:** Pages take a long time to render.

**Solutions:**
1. Check your internet speed
2. Reduce the date range for reports/analytics
3. Close unused browser tabs
4. Try a different browser
5. Disable browser extensions temporarily

### Browser Crashes

**Symptoms:** Browser becomes unresponsive on Sparkwave.

**Solutions:**
1. Update your browser to the latest version
2. Clear browser cache and storage
3. Disable hardware acceleration
4. Check available RAM on your device
5. Try a different browser

---

## Account & Billing

### Can't Upgrade Plan

**Symptoms:** Upgrade button doesn't work or payment fails.

**Solutions:**
1. Check that your payment method is valid
2. Ensure your card isn't blocked for online purchases
3. Try a different payment method
4. Contact your bank if the card is declined
5. Reach out to support for manual assistance

### Missing Features After Upgrade

**Symptoms:** New features don't appear after upgrading.

**Solutions:**
1. Log out and log back in
2. Clear browser cache
3. Wait 5-10 minutes for changes to propagate
4. Contact support if features still don't appear
