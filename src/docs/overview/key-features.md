# Key Features

Sparkwave comes packed with features designed to automate your business operations. Here's what's included:

## 🤖 AI Agents

Intelligent agents that handle specific business functions:

- **Lead Follow-up Agent** — Automatically follows up with new leads via SMS/email
- **Booking Reminder Agent** — Sends appointment reminders and handles confirmations
- **Review Request Agent** — Asks happy customers for reviews at the right time
- **Reactivation Agent** — Re-engages dormant customers with personalized outreach

[Learn more about Agents →](/docs/features/agents)

## 📋 Task Management

Track and manage all automated tasks:

- View pending, completed, and failed tasks
- Manual override when AI needs human input
- Task history and audit trail
- Priority queuing for time-sensitive actions

[Learn more about Tasks →](/docs/features/tasks)

## 🏥 Health Monitoring

Real-time system health visibility:

- Agent status indicators (healthy/warning/critical)
- Integration connection status
- Error rate tracking and alerts
- Performance metrics and trends

[Learn more about Health Monitoring →](/docs/features/health-monitoring)

## 📊 Reports & Analytics

Understand your automation ROI:

- Daily/weekly/monthly activity summaries
- Conversion tracking (leads → customers)
- Agent performance comparisons
- Custom date range reports

[Learn more about Reports →](/docs/features/reports)

## 💬 Rico AI Assistant

Your personal AI helper:

- Ask questions in natural language
- Get instant status updates
- Run ad-hoc reports
- Troubleshoot issues with guidance

[Learn more about Rico →](/docs/features/rico-chat)

## 📧 Email Marketing

Built-in email campaign management:

- Template library with proven designs
- Segmented audience targeting
- A/B testing capabilities
- Performance analytics

## 👥 Contact Management

Centralized customer database:

- Unified contact profiles
- Interaction history
- Segmentation and tagging
- Import/export capabilities

## 🔌 Integrations

Connect your existing tools:

- **Wix** — Sync contacts, bookings, and payments
- **Stripe** — Payment processing and subscription management
- **Google Calendar** — Two-way calendar sync
- **SMS/Email** — Outbound communication channels

## 🎯 Mission Control

Your command center for everything:

- Activity feed showing real-time updates
- Quick actions for common tasks
- At-a-glance status overview
- Customizable dashboard widgets
