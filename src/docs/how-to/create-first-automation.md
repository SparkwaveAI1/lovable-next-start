# Create Your First Automation

Get started with Sparkwave automation in just a few steps.

## Overview

This guide walks you through creating your first automated workflow. By the end, you'll have a working automation that saves you time on repetitive tasks.

## Prerequisites

Before you begin:
- Complete the [onboarding process](/docs/getting-started/onboarding)
- Have at least one integration connected
- Know what task you want to automate

## Step 1: Identify the Task

Start by identifying a repetitive task in your business. Good candidates for automation include:

- **Follow-up emails** - Send automated responses to inquiries
- **Appointment reminders** - Notify customers before their bookings
- **Status updates** - Keep team members informed of progress
- **Data entry** - Automatically log information from forms

## Step 2: Open Mission Control

Navigate to **Mission Control** from the sidebar. This is your command center for all automations.

## Step 3: Create a New Task

1. Click the **"+ New Task"** button
2. Give your automation a descriptive name
3. Select the trigger type (scheduled, event-based, or manual)
4. Configure the action to perform

## Step 4: Configure the Trigger

Choose when your automation should run:

| Trigger Type | Use Case |
|--------------|----------|
| **Scheduled** | Run at specific times (daily reports, weekly summaries) |
| **Event-based** | React to actions (new contact, booking made) |
| **Manual** | Run on-demand when you need it |

## Step 5: Define the Action

Specify what happens when the automation triggers:

1. Select the action type (send email, create task, update record)
2. Configure the action details
3. Add any conditions or filters
4. Preview the automation

## Step 6: Test Your Automation

Before going live:

1. Click **"Test Run"** to simulate the automation
2. Review the results in the activity log
3. Make adjustments if needed

## Step 7: Activate

Once you're satisfied:

1. Toggle the automation to **Active**
2. Monitor the first few runs in the activity log
3. Adjust as needed based on results

## Tips for Success

- **Start simple** - Begin with one straightforward automation
- **Monitor results** - Check the activity log regularly at first
- **Iterate** - Refine your automations based on feedback
- **Ask Rico** - Use the AI assistant for suggestions and troubleshooting
